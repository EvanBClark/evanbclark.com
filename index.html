Pool Lane Tracking App

I developed this web app (using HTML, CSS, and JavaScript) and implemented it at multiple YMCA locations where it was used for over a year during the COVID-19 pandemic. The pandemic brought with it new needs and requirements. Each swimmer had to reserve a lane in the swimming pool and this app helped us track which lanes already had swimmers assigned. It was designed to be as simple and user friendly as possible, all in one HTML file.

Live LED BART Map

I created a live BART (Bay Area Rapid Transit) map out of LED lights using BART’s API. Each train is represented by a light of that train’s line color. I also added a feature where you can record the data over time and play it back like a timelapse.

Online Multiplayer Equatos Game

Equatos is a fun and educational game I created that uses a standard 52-card deck and math equations to score points. This is a multiplayer game that can be played online with another device or with 2 players on the same device. Just open it in two tabs to test out the public online matches.

Blackjack

This is a game of blackjack or 21. You play against an AI dealer that behaves just like a real dealer would. You have complete control over 10 different rules like the number of decks, insurance, and surrender, so that you can customize the game exactly to your liking.
